### 👤 Valeh Agayev — Principal DBA

> *"Reliable data infrastructure is invisible. You only notice it when it's gone."*

Database Administrator with **15+ years** of hands-on experience designing, operating, and recovering enterprise database systems across cloud-native and on-premises environments. Currently at **IKTEX LLC, Baku, Azerbaijan**.

Specialising in:
- 🏗️ **High Availability & DR** — Oracle RAC + Data Guard, SQL Server Always On AG, Patroni, Galera, CloudNativePG
- ⚡ **Performance Engineering** — execution plan analysis, wait event tuning, index strategy, query optimisation
- ☸️ **Cloud-Native Databases** — CloudNativePG on Kubernetes, GitOps-driven DBaaS, PgBouncer connection pooling
- 🤖 **Automation** — Ansible, Terraform, pgBackRest, RMAN, shell scripting, GitHub Actions CI
- 🔐 **Security & Compliance** — least privilege, TDE, pgaudit, Vault integration, hardening standards
