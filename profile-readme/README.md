### 👋 About Me

I'm **Valeh Agayev**, a **Principal / Staff Database Engineer** based in 🇦🇿 Baku, Azerbaijan, with 15+ years of experience building and operating large-scale database platforms.

I specialize in:
- 🏗️ **HA/DR architecture** — RAC, Data Guard, Always On AG, Patroni, Galera, CNPG
- ⚡ **Performance engineering** — query tuning, index design, wait analysis, execution plans
- ☸️ **Cloud-native databases** — CloudNativePG on Kubernetes, GitOps-driven DBaaS
- 🤖 **Automation** — Ansible, Terraform, pgBackRest, RMAN, SQL Agent, shell scripting
- 🔐 **Security** — least privilege, TDE, pgaudit, Vault integration, hardening standards
