### 👋 About Me

I'm **Valeh Agayev**, a **Principal Database Engineer** at IKTEX LLC in Baku, Azerbaijan, with 15+ years of experience building and operating large-scale database platforms across cloud-native and on-premises environments.

I specialize in:
- 🏗️ **HA/DR architecture** — RAC, Data Guard, Always On AG, Patroni, Galera, CNPG
- ⚡ **Performance engineering** — query optimization, index design, wait analysis, execution plans
- ☸️ **Cloud-native databases** — CloudNativePG on Kubernetes, GitOps-driven DBaaS
- 🤖 **Automation** — Ansible, Terraform, pgBackRest, RMAN, SQL Agent, shell scripting
- 🔐 **Security** — least privilege design, TDE, pgaudit, vault integration, hardening
